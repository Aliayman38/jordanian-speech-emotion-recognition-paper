\# Jordanian Speech Emotion Recognition



This repository contains the implementation and research work for the paper:



\*\*Speech Emotion Recognition for the Jordanian Arabic Dialect\*\*



---



\## Overview



This project focuses on recognizing human emotions from speech in the \*\*Jordanian Arabic dialect\*\* using machine learning and deep learning techniques.



The system combines:



\* MFCC acoustic features

\* Wav2Vec2 deep speech embeddings

\* Multiple machine learning classifiers



The goal is to evaluate different approaches for \*\*speech emotion recognition (SER)\*\* and compare their performance on Jordanian Arabic speech.



---



\## Models Implemented



The following models were trained and evaluated:



\* Support Vector Machine (SVM)

\* K-Nearest Neighbors (KNN)

\* Multi-Layer Perceptron (MLP)

\* Wav2Vec2-based classifier



---



\## Dataset



The dataset contains Jordanian Arabic speech samples labeled with the following emotions:



\* Happy

\* Sad

\* Angry

\* Neutral



⚠️ Note:

The dataset is \*\*not included\*\* in this repository because it contains private voice recordings.
